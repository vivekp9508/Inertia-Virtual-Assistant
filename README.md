## 🏗️ How It Works

```text
🎙️ User Speaks  →  🧠 Speech Recognition  →  ⚙️ Command Parsing  →  🔍 Action Decision  →  🔊 Voice Reply / 🌐 Browser Automation
```

1. **Voice Input** — User clicks the orb and speaks a command
2. **Speech Recognition** — Browser converts speech to text using Web Speech API
3. **Command Parsing** — JavaScript checks keywords and intent
4. **Action Trigger** — Assistant decides whether to:
   - respond conversationally,
   - open a browser resource,
   - fetch utility info,
   - or perform a smart search
5. **Speech Output** — Assistant speaks the response using Speech Synthesis API
6. **Live UI Feedback** — terminal response, status indicator, and command counter update instantly
