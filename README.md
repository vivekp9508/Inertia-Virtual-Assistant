# 🎙️ Inertia - Virtual Assistant

A browser-based voice-controlled virtual assistant built using **HTML, CSS, and JavaScript**.
It uses **Speech Recognition** and **Speech Synthesis APIs** to interact with users and perform tasks.


## 🚀 Features

* 🎤 Voice Recognition (Speech-to-Text)
* 🔊 Voice Response (Text-to-Speech)
* 🌐 Open websites (Google, GitHub, Instagram, etc.)
* 🎵 Play songs on YouTube
* ⏰ Tell time and date
* 🧮 Perform basic calculations
* 😂 Tell jokes
* ⏳ Set reminders


## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Web Speech API


## ⚙️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Inertia-Virtual-Assistant.git
```

2. Open folder

3. Run `index.html` in browser
